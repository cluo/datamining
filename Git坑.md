1. SSH连接github 
添加公钥后
```
ssh -T git@github.com
```
更换成ssh方式
```
hunterhug@hunterhug-ThinkPad-T420s ~/git/nima $ git remote set-url origin git@github.com:hunterhug/nima.git
hunterhug@hunterhug-ThinkPad-T420s ~/git/nima $ git remote -v
origin	git@github.com:hunterhug/nima.git (fetch)
origin	git@github.com:hunterhug/nima.git (push)
hunterhug@hunterhug-ThinkPad-T420s ~/git/nima $ 
```
